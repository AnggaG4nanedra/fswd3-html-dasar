HTML (Hyper Text Markup Language)

HTML merupakan bahasa markup standar yang digunakan sebagai dasar pembuatan web. Dengan HTML kita dapat membuat halaman web
yang tersusun sebagai blok menggunakan elemen-elemen HTML seperti tag dan atribut.

Struktur HTML

Sebuah halaman HTML memiliki ekstensi file .html. Dalam sebuah dokumen HTML terdiri dari blok-blok elemen HTML. Sebuah elemen
HTML terdiri dari tag dan atribut HTML. Tag adalah individu dari struktur html, kita harus membuka dan menutup tag dengan garis miring seperti ini.
Ada beberapa variasi dengan tag beberapa di antaranya adalah tag self-closing yang tidak perlu ditutup dan ada juga tag kosong yang bisa kita tambahkan atribut apa saja di dalamnya.
Atribut adalah properti tambahan dari tag html yang mendefinisikan properti dari setiap tag html seperti width, height, control, loop, input, dan autoplay.
Atribut ini juga membantu menyimpan informasi dalam meta tag dengan menggunakan atribut name, content, dan type attributes.

Rangkaian Struktur HTML Sebagai berikut :

  1. !DOCUMENT html : merupakan tag yang digunakan untuk menyimpan informasi versi dari HTML yang dibuat. Secara default versi html yang dimuat adalah versi 5.
  2. html : tag ini merupakan dasar dari elemen html. Tag html merupakan parent tag dari semua element dan tag html, bahkan tag head dan body juga merupakan child dari tag html.
  3. head : tag head digunakan untuk menyimpan informasi meta data, judul sebuah halaman, halaman CSS dan lain-lain. Atribut yang dimuat dalam tag ini tidak akan ditampilkan kepada pengguna. Hal tersebut karena tag yang ditulis dalam tag head digunakan sebagai referensi dan watermark dari penyusun dokumen html.
  4. body : tag body digunakan menampilkan semua data yang meliputi teks, gambar hingga hyperlink. Semua konten yang ada dan ditampilkan pada halaman html akan dimuat di dalam tag ini.
